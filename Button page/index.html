When we use the website and how to use it in emergency cases?

1.If there is a malfunction in button of the elevator ,then a person can access the elevator by using digital button through our website.

At first ,the user have to verify his/her QR code or ID number.After verifying the code the user can enter the website.Then user press the floor number(Where he/she want to go) in the website and after that he/she can reach his/her required floor of that building.

2.If a normal person gets in the elevator with a person in emergency condition ,then the person in emergency can generate a QR code id.

Here Emergency Case means if a woman is pregnant or if a person is handicapped i.e. he/she has broken arms or legs and those who have problems in heart can use the elevator in this way .

For get the access of Emergency case first of all user have to register through given Emergency button or Emergency portal.Then he/she got the access otherwise user can't.

Case 1(one who cannot generate any id)

For that purpose ,the emergency user click the link of the website and open it .Then he/she press the registration button and gets allocation id .Then the user have to verify the id with QR and get access .

Case 2(The user has the id of the website)

Normally in elevator gets in ascending order but in emergency case if the user in emergency verify his/her QR code id and press the floor button .Then this user reaches his/her floor at first .After that the normal person reach his/her required floor .
